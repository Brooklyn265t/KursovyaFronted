Новости

На этой неделе компания Microsoft напомнила, что поддержка Windows 10 закончится 14 октября 2025 года, но предложила расширенную поддержку (Extended Security Updates, ESU), которую можно будет получить за дополнительную плату.

«Хотя мы настоятельно рекомендуем перейти на Windows 11, мы понимаем, что существуют обстоятельства, которые могут помешать вам заменить устройства с Windows 10 до даты прекращения ее поддержки, — пишут представители Microsoft. — Поэтому Microsoft будет предлагать расширенные обновления безопасности».

Как и в случае с программой ESU для Windows 7, которая завершилась 10 января текущего года, пользователи Windows 10 смогут оплатить ежегодную подписку, чтобы продолжать получать ежемесячные обновления безопасности в течение трех лет после окончания поддержки ОС. Стоимость ESU для Windows 10 пока не раскрывается. Однако можно предположить, что условия не будут сильно отличаться от аналогичной программы для Windows 7, которая стоила 50 долларов за одно устройство под управлением Windows 7 Pro в течение первого года; 100 долларов за второй год; и 200 долларов за третий. Для клиентов Windows Enterprise цена была вдвое ниже. В рамках Windows 10 ESU пользователям буду предоставляться только критические и важные обновления безопасности. Исправления для менее значимых недостатков, запросы каких-либо функции или прочие изменения рассматриваться не будут, а техническая поддержка тоже будет ограничена только вопросами безопасности.
